## Tarefa

Criar a página de edição de ticket (/tickets/[id]/edit), carregando os dados existentes do ticket e preparando a base para reutilização do TicketForm.

## Contexto

- Aplicação de Helpdesk (CRUD de Tickets)
- TicketForm já existe e usa:
  - React Hook Form
  - Zod
- API mock disponível:
  - GET /api/tickets/:id
  - PATCH /api/tickets/:id

## Requisitos Técnicos

- Página deve ficar em:
  app/tickets/[id]/edit/page.tsx
- Usar abordagem híbrida:
  - Server Component para buscar o ticket
  - Client Component para o formulário
- Buscar ticket por ID no servidor
- Se o ticket não existir:
  - retornar notFound()
- Passar os dados do ticket como props para o formulário
- NÃO implementar submit ainda
- NÃO implementar toast

## Output Esperado

- Página /tickets/[id]/edit renderizando
- Ticket carregado corretamente
- TicketForm renderizado com valores iniciais
- Tratamento correto de 404
- Código simples e incremental
