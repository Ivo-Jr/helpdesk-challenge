## Tarefa

Criar a página de criação de tickets (/new) com estrutura mínima e funcional.

## Contexto

- Aplicação Next.js (App Router)
- Endpoint disponível: POST /api/tickets
- Página deve ser Client Component
- Não implementar validações avançadas ainda

## Requisitos

- Criar arquivo app/new/page.tsx
- Declarar "use client"
- Renderizar um formulário simples (HTML)
- Submeter dados para POST /api/tickets
- Redirecionar para "/" após sucesso
- Tratar erro básico (console ou mensagem simples)

## O que NÃO fazer

- Não usar React Hook Form ainda
- Não usar Zod ainda
- Não criar TicketForm reutilizável
- Não adicionar toast
- Não aplicar estilos avançados

## Output esperado

- Página /new acessível
- Ticket criado via API
- Redirecionamento funcionando
