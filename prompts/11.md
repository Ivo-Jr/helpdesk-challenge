## Tarefa

Adicionar validações ao formulário de criação utilizando Zod.

## Contexto

- TicketForm já utiliza React Hook Form
- Zod já está configurado no projeto

## Requisitos

- Criar schema de criação de ticket com Zod
- Integrar Zod ao React Hook Form
- Implementar validações obrigatórias:
  - title (min 5)
  - description (min 20)
  - email (formato válido)
  - priority
  - category
- Implementar pelo menos 1 validação condicional:
  - priority = high → description >= 60 chars
    OU
  - category = bug → title contém [BUG]

## Output esperado

- Validações funcionando
- Mensagens de erro exibidas no formulário
